# QR Code Generator

A simple Python project that generates QR codes for any text, URL, or data input.
The generated QR code is saved as an image file that can be scanned using any QR scanner.

## 🚀 Features

* Generate QR codes from text or URLs
* Save QR codes as PNG image files
* Simple and beginner-friendly Python project

## 🛠 Tech Stack

* Python
* qrcode library
* Pillow library

## 📦 Installation

1. Clone the repository

```
git clone https://github.com/AnshikaKumari21/QR-Code-Generator.git
```

2. Navigate to the project folder

```
cd QR-Code-Generator
```

3. Install dependencies

```
pip3 install qrcode[pil]
```

## ▶️ How to Run

Run the Python file in the terminal:

```
python3 main.py
```

After running, a QR code image will be generated and saved in the project folder.

## 📷 Example Output

The program generates a QR code image that can be scanned to access the encoded data.
